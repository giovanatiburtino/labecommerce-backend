## 💻 Funcionalidades do Projeto
- Endpoints
  + GET /users/ 
  + GET /products/ 
  + GET /products/search?q=productName 
  + POST /user/
  + POST /product/ 
  + POST /purchases/
  + GET /products/:id 
  + GET /users/:id/purchases
  + DELETE /users/:id 
  + DELETE /products/:id
